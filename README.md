# STM32 无人机飞控系统
基于 STM32F103C8T6 + FreeRTOS 的四轴无人机飞控系统，包含飞控板代码和遥控器代码。

##  工程结构
STM32-Drone-Flight-Control/
├── Flight/ # 飞控板代码
├── Remote/ # 遥控器代码
└── README.md


### Flight（飞控板）
- 姿态解算、PID 控制、无线接收、电机驱动、电源管理

### Remote（遥控器）
- 摇杆采集、按键检测、无线发送、电源管理

## 🛠️ 技术栈

| 类别 | 技术 |
| 主控 | STM32F103C8T6 |
| RTOS | FreeRTOS（多任务架构） |
| 传感器 | MPU6050（六轴陀螺仪 + 加速度计） |
| 无线通信 | SI24R1（2.4G，SPI 接口） |
| 电源管理 | IP5305T |
| 电机控制 | PWM（定时器输出） |
| 编程语言 | C |
| 开发工具 | Keil MDK、STM32CubeMX、ST-Link |

##  飞控板功能

### 姿态解算
- 使用四元数互补滤波算法融合陀螺仪与加速度计数据
- 输出俯仰角（Pitch）、横滚角（Roll）、偏航角（Yaw）

### PID 控制
- 串级 PID 控制（角度环 + 角速度环）
- 根据欧拉角误差计算电机控制量

### 无线通信
- 基于 SPI 协议驱动 SI24R1
- 自定义数据帧协议（帧头 + 数据 + 校验和 + 帧尾）
- 利用芯片硬件自动应答（ARQ）实现可靠收发
- 数据包长度：18 字节

### FreeRTOS 任务架构
| 任务 | 功能 | 优先级 |
| :--- | :--- | :--- |
| COM_task | 无线接收、连接状态、关机指令、飞行状态 | 4 |
| FLIGHT_task | 姿态解算、PID 计算、电机控制 | 3 |
| POWER_task | 电源保活、关机响应（直接任务通知） | 3 |
| LED_task | LED 状态指示 | 1 |

### 电源管理
- 集成 IP5305T 电源管理芯片
- 通过直接任务通知实现遥控关机指令响应
- 10 秒自动保活，防止芯片休眠

### 安全机制
- 油门序列解锁状态机（满油 1 秒 → 归零 1 秒）
- 失联超时保护（500ms 连续失联进入故障状态）

### LED 状态指示
| 位置 | 状态 | 含义 |
| :--- | :--- | :--- |
| 灯1（左上）+ 灯2（右上） | 常亮 | 遥控器已连接 |
| 灯1 + 灯2 | 熄灭 | 遥控器未连接 |
| 灯3（左下）+ 灯4（右下） | 慢闪（1Hz） | 空闲状态 |
| 灯3 + 灯4 | 快闪（5Hz） | 常规飞行 |
| 灯3 + 灯4 | 常亮 | 定高模式 |
| 灯3 + 灯4 | 熄灭 | 故障状态 |

##  遥控器功能

### 摇杆控制
- ADC + DMA 采样四通道摇杆数据
- 映射范围：0～1000
- 映射关系：
  - thr（油门）：0～1000（上升/下降）
  - pit（俯仰）：0～1000 → -10°～+10°（前进/后退）
  - rol（横滚）：0～1000 → -10°～+10°（左移/右移）
  - yaw（偏航）：0～1000 → 左转/右转

### 按键功能
| 按键 | 功能 |
| :--- | :--- |
| 上/下 | 俯仰微调 ±10 |
| 左/右 | 横滚微调 ±10 |
| 左键 | 关机指令 |
| 右键 | 定高模式切换 |
| 右键长按 | 摇杆零偏校准 |

### 数据发送
- 周期性打包 18 字节数据包（COM_TASK，10ms 周期）
- 帧头（`<<`）+ 10 字节数据 + 4 字节校验和 + 帧尾（`>>`）

##  核心模块说明

### 公共模块（common/）
| 文件 | 功能 |
| :--- | :--- |
| Com_debug | 重定向 `printf`，实现 `debug_printf()` |
| Com_filter | 一阶低通滤波（角速度）+ 卡尔曼滤波（加速度） |
| Com_imu | 四元数互补滤波姿态解算 |
| Com_pid | 串级 PID 计算 |

### 硬件接口层（interface/）
| 文件 | 功能 |
| :--- | :--- |
| Int_IP5305T | 模拟短按开机、保活、关机 |
| Int_motor | PWM 电机控制（支持结构体参数） |
| Int_led | LED 开/关/翻转控制 |
| Int_SI24R1 | 2.4G 收发、寄存器读写、模式切换 |
| Int_mpu6050 | 六轴传感器数据读取 |
| Int_key | 按键状态检测（长短按） |
| Int_joystick | 摇杆 ADC 采集（DMA） |

### 应用层（Application/）
| 文件 | 功能 |
| :--- | :--- |
| App_freeRTOS_Task | FreeRTOS 任务创建与调度 |
| App_flight | 姿态解算 + PID 控制 + 电机输出 |
| App_receive_data | 数据接收、连接状态、解锁逻辑、飞行状态 |
| App_process_data | 摇杆校准、按键处理、数据处理 |
| App_transmit_data | 遥控数据打包发送 |
